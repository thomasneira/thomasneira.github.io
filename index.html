ATP Match Prediction: Elo-Driven Classification

Sports Analytics · Feature Engineering · Temporal Validation

Built a predictive model for ATP tennis match outcomes using 10 years of elite tournament data (2015–2024). The core insight: rather than feeding raw stats into a classifier, I engineered five Elo-derived features capturing relative player strength — both overall and surface-specific — then applied strict temporal validation to ensure no future information leaked into training. A Logistic Regression on those five features outperformed raw-ranking and Elo-only baselines.

0.702 ROC-AUC
