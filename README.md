## Projects

### Lead Conversion Optimization — ExtraaLearn
**Predicting which EdTech leads will convert using machine learning**

Built a full predictive analytics pipeline for an EdTech startup to identify high-probability leads and optimize sales resource allocation.

- **Techniques:** XGBoost, Random Forest, Logistic Regression, statistical hypothesis testing, feature engineering
- **Result:** 86% recall, 0.89 ROC-AUC — projected $847K annual incremental revenue
- **Key finding:** Website-first leads convert 4.3× more than mobile app leads (χ² = 612.4, p < 0.001)
